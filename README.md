# Event Booking API Dökümantasyonu

Bu proje bir etkinlik rezervasyon sistemini simüle etmektedir. Projeyi geliştirme amacım deneyim kazanmaktır.

## Stack

Framework : .NET 10 Web API  
Database : SQL Server  
Architecture : Clean Architecture  
Securitry : JWT  
Caching : Redis & In Memory Caching  
Mail : FluentEmail  

## Bu projeyi geliştirme amacım aşağıdaki konuları tecrübe etmek içindi.

- Eş Zamanlılık Yönetimi (Concurrency-Race condition)
- Performans Yönetimi (Caching - Redis - Docker)
- Arkaplan Görev Yönetimi (Background Services)
- E-mail Gönderme
- API geliştirirken Response Wrapper, Global Exception Handling, Scala UI, Postman kullanımı gibi konuları öğrenmek
- DTO ihtiyacını öğrenmek
- AutoMapper kullanımı
- FluentValidation Kullanımı
- Clean Architecture mantığı
- Unit of Work tasarım deseni kullanım senaryosu
- Repository Pattern ve Generic Repository kullanımı
